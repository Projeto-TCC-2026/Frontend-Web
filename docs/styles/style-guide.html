Style Guide do Sistema de
Monitoramento Pós-Cirúrgico

Padrão visual único para web e mobile, construído para leitura rápida de dados clínicos, hierarquia clara de urgência e confiança visual — do dashboard do profissional de saúde ao app do paciente.

04 · Fundamentos

Tipografia

Três famílias, cada uma com um papel funcional específico — pensadas para leitura clínica rápida, não decoração.

Manrope Display · Títulos e navegação
Geométrica, moderna e com peso forte nos títulos (700/800) — dá personalidade à marca sem parecer "corporativo genérico". Usada em H1, H2, H3, nomes de seção e labels de navegação.
Inter Corpo de texto · Interface
Desenhada especificamente para telas, com ótima legibilidade em tamanhos pequenos. Usada em parágrafos, labels de formulário, botões e todo o texto de interface.
IBM Plex Mono Dados vitais · Valores numéricos
Números tabulares monoespaçados — essencial para leitura rápida de sinais vitais (BPM, SpO2, pressão), timestamps e códigos de paciente, onde alinhamento e não-ambiguidade importam mais que estética.

05 · Componentes

Estados de input

Todos os inputs seguem o mesmo comportamento: borda neutra em repouso, borda azul + glow suave no foco, e cor semântica para validação. Transições de 150ms em todos os estados.
